# gem5 to rem6 Alignment

This file is a compatibility index for older plans and review gates that refer
to `docs/architecture/gem5-to-rem6-alignment.md`.

It does not own migration scores, checklists, proof logs, or component status.

## Boundary

- `docs/architecture/rem6-architecture.md` owns the stable architecture story:
  runtime shape, ownership model, invariants, and design motivation.
- `docs/architecture/gem5-to-rem6-migration.md` owns current gem5-to-rem6
  migration state, component scoring, executable evidence, missing behavior,
  and next-evidence notes.

Keep this file short. When implementation evidence changes, update the
migration ledger. When the stable architecture changes, update the architecture
overview.
